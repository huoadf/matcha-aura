# Matcha Particle & Spark Aura System

A lightweight, high-performance particle & spark 3D aura system for **Matcha LuaVM** (Roblox external script).

## 🚀 Execution Loadstring

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/huoadf/matcha-aura/main/aura.lua"))()
```

## ✨ Features
- **High Performance**: Pre-allocated Drawing circle pool (runs at 60-120 FPS).
- **Orbiting Main Particles**: Dynamic wave-pulsing particles surrounding your character.
- **Spark Energy Orbs**: Fast orbiting sparks with configurable glow intensity.
- **INS-UI Interface**: Interactive menu window with sliders for sizes, counts, speeds, opacity, and color pickers.
- **External Global API**: `_G.set_aura_color`, `_G.set_aura_enabled`, `_G.get_aura_config`.
